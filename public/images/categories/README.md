# Category Thumbnail Images

Place your category thumbnail images in this folder with the following exact filenames:

## Required Images

| Filename | Category | Description |
|----------|----------|-------------|
| `PEDESTRIAN CROSSING SXC.jpg` | Lampu Penyebrangan | Pedestrian crossing lights |
| `TRAFFIC LIGHT.jpg` | Traffic Light | Traffic signal lights |
| `ALAT PENERANGAN JALAN.jpg` | Alat Penerangan Jalan | Street lighting equipment |
| `WARNING LIGHT.jpg` | Warning Light | Warning/safety lights |

## Image Specifications

- **Resolution**: 2448 x 3264 pixels (portrait orientation)
- **Format**: JPG
- **File names**: Must match exactly (case-sensitive)

## Important Notes

1. File names must match exactly as shown above
2. The system uses these exact paths:
   - `/images/categories/PEDESTRIAN CROSSING SXC.jpg`
   - `/images/categories/TRAFFIC LIGHT.jpg`
   - `/images/categories/ALAT PENERANGAN JALAN.jpg`
   - `/images/categories/WARNING LIGHT.jpg`

3. After placing the images, restart the development server if running
